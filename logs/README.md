# Logs Folder

This directory contains all interaction logs between the user and Smart Wellness Planner+.

| File | Description |
|------|--------------|
| session_2025-10-29_normal_mode.json | Interactions when the AI was in Normal mode |
| session_2025-10-29_exam_mode.json | Interactions when the AI was in Exam mode |

Each log stores:
- Timestamp  
- User input text  
- Model-generated nutrition & insight  
- Active mode (Exam / Normal)
